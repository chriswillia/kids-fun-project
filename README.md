# Math Mission

A colourful browser math game for kids to practise the four core
operations: **addition, subtraction, multiplication, and division**.

## How to play

1. Open `index.html` in a browser.
2. Pick which operations to practise (+, −, ×, ÷) and a difficulty level.
3. Choose how many questions you want in the round.
4. Type each answer and press **Check** (or hit Enter).
5. Streaks earn bonus points - how high can you go?

## Difficulty levels

| Level  | Number range | Times/divide cap |
|--------|--------------|------------------|
| Easy   | 0-10         | 10               |
| Medium | 0-20         | 12               |
| Hard   | 1-50         | 15               |

Subtraction problems always have a non-negative answer, and division
problems always divide cleanly into a whole number.

## Files

- `index.html` - game layout and screens (setup, quiz, results)
- `style.css` - kid-friendly styling with colourful feedback
- `script.js` - problem generator, scoring, and streak logic
