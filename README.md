# Kids Fun Project

A simple, colorful balloon pop game that kids can play in the browser.

## How to use

1. Open `index.html` in a browser.
2. Click the balloons to pop them.
3. Watch your score grow!

## Files

- `index.html` - game layout and content
- `style.css` - bright and playful styling
- `script.js` - balloon game logic

## Notes

This is a great starter project for learning HTML, CSS, and JavaScript. You can extend it by:
- adding sound effects
- adding a timer
- showing a win animation when all balloons are popped
